# BetterNCM-v2
### PC版网易云客户端插件管理器
![Microrange](https://github.com/MicroCBer/BetterNCMPlugins/raw/main/Stylesheets/Theme-Microrange/white.png)
最新测试兼容版本：`2.9.9`

# 安装
## Windows
**若安装了BetterNCM v1 请先卸载**
将release的`msimg32.dll`放入网易云音乐安装目录中即可
